# GameEngine
Advanced Programming Coursework
C++ Game Engine created in 3rd year of University by Chris Bodsworth

Middleware used:
-Graphics: Irrlicht
-Physics: PhysX
-UI: IMGUI
-Keyboard input: SFML
-Sound: IrrKlang
-Data Input: Lua

Music sourced from Bensound.
Assets either made or sourced from sketchfab.
